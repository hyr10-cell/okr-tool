# OKR 목표관리 시스템 기획서
> CLAP(clap.company) 화면 기반 자사 사내 툴 구현
> 대상: 50~60명 규모 단일 조직 / 사내 전용
> 작성일: 2026-04-17

---

## 1. 서비스 개요

### 1.1 목적
조직 내 구성원·팀·회사 단위의 목표를 수립·추적하고, 피드백을 통해 구성원 간 소통을 강화하는 사내 통합 성과관리 플랫폼.

### 1.2 핵심 가치
- **얼라인(Align)**: 회사-팀-개인 목표를 사이클 단위로 정렬
- **트래킹(Track)**: 체크인으로 목표 진행 상황 실시간 관리
- **소통(Communicate)**: 피드백으로 구성원 간 건강한 소통 문화 형성
- **인사이트(Insight)**: 성장 대시보드를 통한 개인·팀 성과 데이터 가시화

### 1.3 사용자 역할
| 역할 | 설명 |
|------|------|
| 관리자 (Admin) | 전사 설정, 권한 관리, 인원 관리 전반, 모든 데이터 접근 |
| 매니저 (Manager) | 팀원 목표 조회, 피드백, 멤버 대시보드 접근, 목표 승인 |
| 멤버 (Member) | 자신의 목표 수립·체크인, 피드백 송수신 |

(이 문서는 전체 기획서이므로 나머지 내용은 원본을 그대로 가져옵니다.)
